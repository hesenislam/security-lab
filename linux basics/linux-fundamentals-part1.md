# Linux Fundamentals Part 1

## Goal
Understand basic Linux commands used in cybersecurity environments.

## Commands Practiced
- pwd – shows current working directory
- ls – lists files and directories
- cd – changes directory
- cat – displays file contents
- whoami – shows current user
- uname -a – displays system information

## Why This Matters
Linux is the backbone of most servers and security tools. Knowing basic navigation and system commands is essential for both offensive and defensive security roles.
