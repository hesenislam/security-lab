# Linux Fundamentals Part 2

## Topics Covered
- File permissions (r, w, x)
- chmod command
- Users and groups
- Basic file operations

## Key Commands
- ls -l
- chmod
- whoami
- groups
- touch, mkdir, cp, mv, rm

## Security Importance
Misconfigured permissions can lead to unauthorized access and privilege escalation. Understanding Linux permissions is critical for system security.
