# security-lab
This repository contains my cybersecurity labs, practice exercises, and scripts while preparing for CompTIA Security+.
## Projects:
- Linux basics lab
- Networking fundamentals practice
- Python scripts for security (port scanner, log analyzer)
- TryHackMe & OverTheWire write-ups
## Goals
- Build strong fundamentals in cybersecurity
- Prepare for the CompTIA Security+ certification
- Develop hands-on skills in Linux, networking, and Python
- Create a practical cybersecurity portfolio

## Status
This repository is actively maintained and updated as I continue learning and practicing cybersecurity concepts.
